<INPUT>

The input tag is an HTML element used to create interactive form controls within a web page. It allows users to input various types of data such as text, numbers, dates, checkboxes, radio buttons, and more.It is a Empty tag The input tag can have several attributes to specify its behavior, appearance, and functionality. Here are some commonly used attributes:
type: This attribute specifies the type of input control to be displayed. Examples include:
text: Creates a single-line text input field.
password: Creates a password input field where the entered text is masked.
number: Creates a numeric input field.
date: Creates a date input field.
checkbox: Creates a checkbox for multiple choice options.
radio: Creates a radio button for selecting a single option from multiple choices.
Many other types are available. The type attribute determines the specific behavior.
name: This attribute assigns a name to the input control, which is used to identify the data when the form is submitted to the server.
value: This attribute specifies the initial value of the input control. It can be used to pre-fill the input field with a default value.
placeholder: This attribute provides a short hint or example text that is displayed in the input field before the user enters a value.
required: When added to the input tag, this attribute specifies that the input field must be filled out before submitting the form.
disabled: This attribute disables the input control, preventing the user from interacting with it or entering any data.
size: This attribute specifies the width of the input field in characters. It can be used to set the visible width of the input control.
maxlength: This attribute specifies the maximum number of characters allowed in the input field.
min and max: These attributes are used for input types like number and date to define the minimum and maximum allowed values, respectively.

<TEXTAREA>

The <textarea> element is an HTML tag used to create a multiline text input area within a web page. It allows users to enter and edit larger amounts of text. The <textarea> tag can have several attributes to specify its behavior, appearance, and functionality. Here are some commonly used attributes:
name: This attribute assigns a name to the textarea element, which is used to identify the data when the form is submitted to the server.
rows and cols: These attributes define the visible number of rows and columns in the textarea, respectively. The rows attribute determines the vertical size, and the cols attribute determines the horizontal size.
placeholder: This attribute provides a short hint or example text that is displayed in the textarea before the user enters any content.
required: When added to the <textarea> tag, this attribute specifies that the textarea field must be filled out before submitting the form.
disabled: This attribute disables the textarea, preventing the user from interacting with it or entering any text.
readonly: When added, this attribute makes the textarea read-only, allowing the user to view the text but not modify it.
maxlength: This attribute specifies the maximum number of characters allowed in the textarea.

<SELECT>

The <option> tag is an HTML element used within the <option> element to define individual options in a dropdown list or selection menu. It allows users to choose one or more options from a predefined set of choices. The <option> tag can have several attributes to specify its behavior and appearance. Here's a breakdown of the important attributes:
value: This attribute specifies the value associated with the option. When the form is submitted, the selected value of the option is sent to the server or can be accessed via JavaScript. If the value attribute is not specified, the text content of the <option> tag will be used as the value.
selected: When added to an <option> tag, this attribute specifies that the option should be pre-selected when the page loads or the dropdown is rendered. Only one option should have the selected attribute within a given <option> element.
disabled: This attribute disables the option, preventing the user from selecting it. Disabled options are typically displayed grayed out and cannot be interacted with.
label: This attribute provides a label or descriptive text for the option. It is often used for accessibility purposes to provide additional information about the option.
